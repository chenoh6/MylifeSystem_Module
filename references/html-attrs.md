# HTML 属性 → 自然语言映射表

## 按功能分类

### 1. 文本/内容属性
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `innerHTML` | 内部HTML内容 | 元素里面的所有HTML标记 |
| `innerText` | 内部文本 | 元素里面的人类可读文字（忽略HTML标签） |
| `textContent` | 纯文本内容 | 元素的原始文本（包含隐藏文本） |
| `value` | 当前值 | 表单输入框/下拉框当前填入的内容 |
| `placeholder` | 占位提示 | 输入框为空时显示的灰色提示文字 |
| `alt` | 替代文字 | 图片无法显示时出现的文字；屏幕阅读器的描述 |
| `title` | 标题提示 | 鼠标悬停时显示的工具提示 |
| `label` | 标签文字 | 表单字段的说明文字 |

### 2. 资源/链接属性
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `src` | 资源来源 | 图片/脚本/音频/视频等的来源URL |
| `href` | 链接地址 | 点击跳转的目标URL |
| `action` | 提交地址 | 表单提交的服务器URL |
| `data` | 数据地址 | 自定义数据属性的基础名 |
| `poster` | 封面图片 | 视频加载时显示的预览图 |
| `icon` | 图标资源 | 网站/应用的图标 |

### 3. 样式/外观属性
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `class` | 样式类 | 关联的CSS类名（多个用空格分隔） |
| `style` | 行内样式 | 直接写在元素上的CSS样式 |
| `id` | 唯一标识 | 元素的唯一ID（用于JS/CSS定位） |
| `color` | 文字颜色 | 前景色/文字色 |
| `background` | 背景样式 | 背景色或背景图 |
| `width` | 宽度 | 元素的水平尺寸 |
| `height` | 高度 | 元素的垂直尺寸 |
| `display` | 显示模式 | 块级/行内/弹性等显示方式 |
| `visibility` | 可见性 | 显示/隐藏（保留空间） |
| `opacity` | 不透明度 | 0-1的透明度值 |

### 4. 交互/行为属性
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `onclick` | 点击事件 | 鼠标点击时执行的代码 |
| `onchange` | 变更事件 | 值改变时执行的代码 |
| `oninput` | 输入事件 | 每次输入时执行的代码 |
| `onmouseover` | 悬停事件 | 鼠标移入时执行的代码 |
| `onkeydown` | 键盘按下 | 键盘按键按下时 |
| `disabled` | 禁用状态 | 是否禁用（禁用则不可操作） |
| `readonly` | 只读状态 | 是否只读（不可编辑但可选中） |
| `required` | 必填项 | 是否为必填字段 |
| `checked` | 选中状态 | 复选框/单选框是否勾选 |
| `selected` | 选中状态 | 下拉选项是否默认选中 |

### 5. 结构/布局属性
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `tagName` | 标签名 | 元素的HTML标签类型（如DIV、SPAN） |
| `parentNode` | 父节点 | 元素的上一层元素 |
| `childNodes` | 子节点们 | 元素下面的所有子元素/文本 |
| `nextSibling` | 下一个兄弟 | 同级的下一个元素 |
| `previousSibling` | 上一个兄弟 | 同级的上一个元素 |

### 6. 表单相关
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `name` | 字段名称 | 表单提交时的参数名 |
| `type` | 输入类型 | 文本/密码/邮箱/文件等输入类型 |
| `form` | 所属表单 | 元素关联的表单ID |
| `method` | 提交方式 | GET/POST 提交方法 |
| `enctype` | 编码类型 | 表单数据的MIME类型 |

### 7. 元信息
| HTML属性 | 自然语言描述 | 语义含义 |
|----------|-------------|----------|
| `charset` | 字符编码 | 文档的字符编码（如UTF-8） |
| `lang` | 语言代码 | 内容的语言（如zh-CN） |
| `viewport` | 视口配置 | 移动端缩放设置 |
| `name` (meta) | 元信息名 | meta标签的名称 |
| `content` (meta) | 元信息内容 | meta标签的具体值 |
